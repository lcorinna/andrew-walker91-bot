# Andrew Walker91 Bot 🤖

Telegram-бот, который отвечает "пизда", если кто-то пишет "да" или "da" — только как отдельное слово (в разных регистрах).
Создан в честь легендарного **@andrew_walker91**, который отвечает так в каждом чате.

---

## 🔧 Возможности

- 💬 Строго реагирует на сообщение с **"да"** или **"da"** (учитывает регистр: ДА, дА и т.д.)
- 🎲 Отвечает фразой, случайной картинкой, гифкой или аудио
- 📌 Отправляет сообщение **как ответ (reply)** на оригинальное сообщение
- 🔁 Поддерживает **редактированные сообщения**
- 💘 **Ставит реакцию** на каждое _N_-ое сообщение в чате
- 📊 Поддерживает подробную статистику по чатам (хранится в кэше для скорости)
- 📅 **Автоматический отчет:** каждый месяц отправляет статистику админу в личку

---

## 💘 Реакции

Бот выбирает случайное число от `100` до `300` индивидуально для каждого чата. Когда общее количество написанных сообщений в чате достигает этого числа, бот:

1. Ставит реакцию **💘** на текущее сообщение.
2. Сбрасывает счётчик.
3. Выбирает новое случайное число для следующей реакции.

> 🔐 **Важно:** Для работы кастомных реакций через API бот должен быть назначен администратором в группе.

## 🛠 Техническая часть и установка

Бот написан на Node.js с использованием `node-telegram-bot-api`.
Логика обработки сообщений вынесена в отдельный модуль для удобства поддержки. Статистика кэшируется в оперативной памяти и сохраняется на диск (файл `stats.json`) каждые 5 минут, чтобы избежать лишней нагрузки на файловую систему.

### Запуск проекта:

1. Склонируйте репозиторий и перейдите в папку.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Создайте файл .env в корне проекта и добавьте токен вашего бота:
   ```bash
   BOT_TOKEN="Ваш_Токен_От_BotFather"
   ADMIN_ID=ваш_telegram_id
   PORT=3000
   ```
4. Запустите бота:
    ```bash
    node index.js
    ```
💡 Для работы в фоновом режиме на сервере (Linux) рекомендуется использовать команду:
    ```bash
    nohup node index.js & disown
    ```
